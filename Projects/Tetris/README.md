## Input

|Key|Action|
|---|---|
| `←` or `A` | Move Left       |
| `→` or `D` | Move Right      |
| `↓` or `S` | Fall Faster     |
| `Q`        | Spin Left       |
| `E`        | Spin Right      |
| `P`        | Pause or Resume |
| `Enter`    | Confirm         |
| `Escape`   | Close Game      |
