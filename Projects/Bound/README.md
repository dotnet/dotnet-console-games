Bound is a game where you `^` try to reach the `@` without touching the `#`. It is inspired by the "bound" custom games from Starcraft.

```
  Bound

  Lives: 98
  Level: 1
  ╔═══════╗
  ║   @   ║
  ║       ║
  ║##### #║
  ║#     #║
  ║# #####║
  ║#     #║
  ║##### #║
  ║#     #║
  ║# #####║
  ║       ║
  ║       ║
  ╚═══════╝
```

## Input

- `↑`, `↓`, `←`, `→`: movement
- `enter`: confirm
- `escape`: close
